# windows 가상 메모리 주소 공간

## 32비트 Windows의 기본 가상 주소 공간
- 낮은 2GB(0x00000000~0x7FFFFFFF): 프로세스에서 사용됩니다.
- 높은 2GB(0x80000000~0xFFFFFFFF): 시스템에서 사용됩니다.

## 64비트 Windows의 기본 가상 주소 공간
- 낮은 2GB(0x00000000~0x7FFFFFFF): 프로세스에서 사용됩니다.
- 높은 2GB(0x80000000~0xFFFFFFFF): 시스템에서 사용됩니다.



# 프로그램 기본 주소

## EXE 기본주소
- 32비트 0x000400000 
- 64비트 0x140000000

## DLL 기본주소
- 32비트 0x010000000 
- 64비트 0x180000000
